# QR Pro Generator

Professional high-resolution QR code generator with instant customization. No watermarks, no limits.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

The app will be available at `http://localhost:3000`
